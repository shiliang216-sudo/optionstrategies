# 期权策略展示平台 - 技术架构文档

## 1. 技术栈选择

### 前端技术
- **框架**：React 18（使用Vite构建工具）
- **样式**：Tailwind CSS 3（快速开发响应式样式）
- **动画**：CSS动画 + React Transition Group
- **图标**：Lucide React（轻量级图标库）
- **字体**：Google Fonts（Noto Serif SC, Noto Sans SC, JetBrains Mono）
- **部署**：静态单页应用

### 项目初始化
- 使用 Vite 创建 React 项目
- TypeScript 类型支持
- PostCSS 和 Autoprefixer

## 4. 组件设计

### 4.1 布局组件

#### Header（导航栏）
- **功能**：固定顶部，平滑滚动到各区域
- **状态**：滚动后背景模糊效果
- **响应式**：桌面端完整导航，移动端汉堡菜单
- **Props**：`none`

#### Footer（页脚）
- **功能**：版权信息和快速链接
- **样式**：简洁深色背景
- **Props**：`none`

### 4.2 业务组件

#### Hero（首屏区域）
- **功能**：品牌标语展示，核心价值主张
- **内容**：大标题、副标题、CTA按钮
- **动画**：文字渐入，背景动画效果
- **Props**：`none`

#### FourStrategies（四种策略展示）
- **功能**：展示四种基础交易策略
- **数据**：来自content.ts
- **交互**：悬停效果，点击展开详情
- **Props**：`none`

#### Greeks（希腊字母展示）
- **功能**：展示五个希腊字母指标
- **数据**：来自content.ts
- **布局**：响应式卡片网格
- **Props**：`none`

#### OptionBasics（期权基础概念）
- **功能**：期权定义、核心要素、买卖对比
- **内容**：文字说明、对比表格
- **布局**：左右分栏或堆叠布局
- **Props**：`none`

## 7. 动画设计

### 7.1 页面加载动画
- Hero文字依次渐入（stagger: 200ms）
- 背景渐变动画（duration: 10s）

### 7.2 滚动触发动画
- 使用 Intersection Observer API
- 元素进入视口时添加 `.animate-in` 类
- 动画：translateY(20px) → translateY(0), opacity: 0 → 1

### 7.3 交互反馈
- 按钮悬停：scale(1.05), 阴影加深
- 卡片悬停：translateY(-4px), 边框发光
- 链接悬停：颜色渐变，下划线动画

### 7.4 性能优化
- 使用 `transform` 和 `opacity` 动画（GPU加速）
- 避免触发布局的重排动画
- 使用 `will-change` 提示浏览器优化

## 10. 开发规范

### 10.1 命名规范
- 组件名：PascalCase（如 `OptionBasics.tsx`）
- 文件名：kebab-case（如 `option-basics.tsx`）
- 变量名：camelCase
- 常量：UPPER_SNAKE_CASE

### 10.2 代码风格
- 使用TypeScript严格模式
- ESLint + Prettier 配置
- 函数组件 + Hooks
- 组件单一职责

### 10.3 Git提交规范
```
feat: 新功能
fix: 修复bug
style: 代码格式调整
refactor: 重构
docs: 文档更新
```
