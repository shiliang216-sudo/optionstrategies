# 期权策略展示平台 - 产品需求文档

## 1. 产品概述

期权策略展示平台是一个专业的期权知识教育网站，旨在帮助用户理解期权交易的核心概念、希腊字母以及四种基础交易形态。网站采用现代化设计，提供流畅的交互动画和清晰的内容展示，让复杂的金融概念变得易于理解。

**目标用户**：期权初学者、有兴趣学习期权交易的投资者、金融教育机构

**核心价值**：将复杂的期权知识可视化、交互化，降低学习门槛

---

## 2. 核心功能模块

### 2.1 页面结构

1. **首页（Hero区域）**：品牌标语、核心概念导航入口
2. **期权基础模块**：期权定义、核心要素、买卖双方对比
3. **四种基础交易形态**：
   - 买入看涨（Long Call）
   - 卖出看涨（Short Call）
   - 买入看跌（Long Put）
   - 卖出看跌（Short Put）
4. **希腊字母模块**：
   - Delta（方向敏感度）
   - Gamma（Delta变化率）
   - Theta（时间衰减）
   - Vega（波动率敏感度）
   - Rho（利率敏感度）
5. **期权优势与风险**：
   - 期权优势介绍
   - 风险与挑战说明
   - 核心总结

### 2.2 页面详细设计

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero区域 | 品牌标语展示，副标题动画效果 |
| 首页 | 快速导航 | 四个基础策略的快速入口卡片 |
| 期权基础 | 什么是期权 | 通俗易懂的期权定义，带生活化比喻 |
| 期权基础 | 核心要素 | Call/Put、行权价、到期日、权利金详解 |
| 期权基础 | 买方vs卖方 | 对比展示两者的权利义务差异 |
| 四种策略 | 策略卡片 | 点击展开详细说明，包含盈亏分析 |
| 希腊字母 | 五大指标 | 每个指标独立的卡片，带图标和实例 |
| 总结 | 进阶建议 | 学习路径指导和建议 |

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：金融专业风格，采用深色主题配合金色/绿色点缀
- **风格**：简洁大气、专业权威、现代化
- **质感**：玻璃拟态效果，渐变背景，几何装饰元素

### 4.2 色彩方案
```
主色：#1a1a2e（深蓝黑）
次色：#16213e（深海蓝）
强调色：#e94560（玫红色 - 用于CTA按钮和重点）
辅助色：#0f3460（靛蓝色 - 用于卡片背景）
金色点缀：#ffd700（用于图标和装饰）
绿色：#00ff88（用于上涨/正向指标）
红色：#ff4757（用于下跌/风险提示）
文字主色：#ffffff
文字次色：#a0a0a0
```

### 4.3 字体选择
- **标题**：Noto Serif SC（思源宋体）- 优雅权威
- **正文**：Noto Sans SC（思源黑体）- 清晰易读
- **数字/代码**：JetBrains Mono - 专业感

### 4.4 布局方式
- **桌面端**：1200px最大宽度，响应式网格布局
- **移动端**：全宽布局，堆叠式设计
- **间距**：8px基准网格系统
- **卡片圆角**：16px
- **阴影**：多层阴影营造深度感

### 4.5 图标风格
- 使用emoji作为主要图标风格（符合目标网站风格）
- 金融相关emoji：📈📉💰📊🎓⚠️⏰💎🏆🎯
- 箭头emoji：→ ← ↑ ↓

### 4.6 响应式断点
```
桌面端：> 1024px（3-4列布局）
平板端：768px - 1024px（2列布局）
移动端：< 768px（单列布局）
```
